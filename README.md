# 🚀 Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript**.  
This project showcases my skills, projects, and interests in a clean and visually engaging way.

---

## 📌 Features

- Responsive design (desktop & mobile friendly)
- Smooth scrolling and basic animations
- Skills section with icons
- About & introduction section
- Clean UI with minimal design
- Easy to customize and deploy

---

## 🛠️ Tech Stack

- **HTML5** – Structure and layout  
- **CSS3** – Styling and responsiveness  
- **JavaScript (Vanilla JS)** – Interactivity and logic

## 🌐 Deployment

You can deploy this portfolio easily using:

- **GitHub Pages**
- **Netlify**
- **Vercel**

### Example: Deploying with GitHub Pages

1. Push the code to your GitHub repository
2. Go to **Settings → Pages**
3. Select the branch and root directory
4. Save and your site will be live 🎉
